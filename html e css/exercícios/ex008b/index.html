Hipertexto é todo texto inserido em um documento para web e que tem como principal característica a possibilidade de se interligar a outros documentos da web com uso dos nosos já conhecidos links, presentes nas páginas dos sites que estamos acostumados a visitar. Então, todo o conteúdo textual que você vê em uma página de um site é um hipertexto, assim como imagens, vídeos, gráficos, sons e conteúdos não textuais em geral são chamados de hipermídia.

A sitação usando BLOCKQUOTE, é usada para uma citação que não ficará em uma linha só, ela será usada para citações maiores no site ficará disposta de uma maneira diversa a citação simples. Podemos usar ainda o CITE dentro da referida tag para dar a parte interna do site, os devidos direitos autorais para o criador da citação que você usou.
